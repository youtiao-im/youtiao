# channels.subscribe

This method subscribes to a channel.


## URL

`POST https://api.youtiao.im/v1/channels/:id/subscribe`


## Arguments

| Argument | Example | Required | Description       |
| -------- | ------- | -------- | ----------------- |
| `id`     | `1`     | Required | Channel id        |


## Response

```json
```


## Errors

| Error | Description     |
| ----- | --------------- |
| `404` | No such channel |
