# channels.show

This method gets information about a channel.


## URL

`GET https://api.youtiao.im/v1/channels/:id`


## Arguments

| Argument | Example | Required | Description       |
| -------- | ------- | -------- | ----------------- |
| `id`     | `1`     | Required | Channel id        |


## Response

```json
```


## Errors

| Error | Description     |
| ----- | --------------- |
| `404` | No such channel |
