# feeds.index

This method lists all feeds in a channel.


## URL

`GET https://api.youtiao.im/v1/channels/:channel_id/feeds`


## Arguments

| Argument     | Example | Required | Description         |
| ------------ | ------- | -------- | ------------------- |
| `channel_id` | `1`     | Required | Id of the channel   |


## Response

```json
```


## Errors

| Error | Description        |
| ----- | ------------------ |
| `400` | Invalid parameters |
| `404` | No such channel    |
